# Sky Take-Out 苍穹外卖

## 项目概述
Sky-Take-Out 是一个基于Spring Boot的外卖管理系统，主要用于餐厅员工管理、菜品管理和订单处理。

## 技术栈
- **后端框架**: Spring Boot
- **数据库**: MySQL
- **构建工具**: Maven
- **项目结构**: 多模块架构

## 项目结构
- sky-take-out/ 
- ├── sky-common/ # 公共模块 
- ├── sky-pojo/ # 实体类和数据传输对象 
- └── sky-server/ # 业务逻辑和控制器
